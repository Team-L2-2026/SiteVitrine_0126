Version Démo Statique

Ceci est une version de démonstration statique. Le projet complet utilise PHP 8+ et MySQL avec une architecture MVC.

Pour tester la version fonctionnelle :

  1. Téléchargez tous les fichiers depuis le dépôt GitHub
  2. Importez la base de données
    mysql -u root -p techsolutions_db < database/techsolutions_db.sql
  3. Configurez un serveur web avec PHP (XAMPP, WAMP, MAMP)
  4. Accédez aux interfaces :
    /client/index.php /admin/dashboard.php
